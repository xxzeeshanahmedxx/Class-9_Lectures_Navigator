# Class 9 Study Hub

Deploy the contents of this `site` folder to Cloudflare Pages.

Data is stored in `/data/*.json`. To update videos later, edit the subject JSON files without touching `index.html`.

Routes:

- `/math`
- `/math/ch1`
- `/math/ch1/ex1.1`
- `/physics/ch1`
- `/chemistry/ch1`
- `/urdu/ch1`
- `/computer/ch1`

Cloudflare Pages uses `_redirects` so direct links work.
